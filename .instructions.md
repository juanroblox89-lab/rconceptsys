## Project Context
- **Project**: Creative Production OS - Agencia RConcept
- **Stack**: Vanilla JS/HTML/CSS + Vite + **Supabase**
- **Deployment**: Vercel
- **AI Integration**: Anthropic Claude 3.5 Sonnet API

## Architecture Overview
```
Frontend:  Vanilla JS (Vite bundler, port 5173)
Backend:   Supabase (PostgreSQL + Auth + Storage)
Hosting:   Vercel (auto-deploy)
API:       /api/chat.js (Anthropic API integration)
```

## Directory Structure
- `/pages/` - Feature modules (dashboard, clients, assignments, billing, etc.)
- `/components/` - UI components (Header, Sidebar, Table, CommandPalette)
- `/services/` - Supabase service integrations
- `/js/` - Core app logic (router, store, app initialization)
- `/api/` - Serverless functions (Claude AI chat)
- `/utils/` - Helper functions
- `/css/` - Styling (responsive design)
- `/supabase/` - Supabase configuration & schema
- `/firebase/` - DEPRECATED (legacy, can be removed)

## Common Tasks
- **Dev**: `npm run dev` (starts on http://localhost:5173)
- **Build**: `npm run build` (production bundle)
- **Clean**: `npm run clean` (clear cache)
